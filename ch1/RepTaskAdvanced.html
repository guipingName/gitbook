发布

发布

  可以使用Gitbook.com来托管你的书籍，不过首先需要注册账号(免费和付费)，免费注册后，用户可以选择升级为付费项目，享受更多服务。登录成功以后在用户页面。可以管理书籍和创建新的书籍。 GitBook.com 为每本书籍都创建了一个 Git 项目，并且使用这个 Git 项目来管理书籍源码。另外，GitBook.com 还可以集成 GitHub，所以用户可以将书籍的源码通过 GitHub 上的项目来管理，这样可以使用 GitHub 带来的各种优点，例如： • 其它用户可以 fork • 用户可以点赞，获得更新提醒 • 用户可以贡献自己的内容

说明 gitbook.com暂时无法注册账号和登陆。

绑定域名

Gitbook可以为每本书绑定域名（前提是用户有自己的域名）。在书籍的属性页面找到“Domain Names”。

导出格式

可以输出 HTML、PDF、eBook 等格式的电子书 生成 PDF 格式：gitbook pdf ./ ./mybook.pdf 生成 epub 格式 gitbook epub ./ ./mybook.epub 生成 mobi 格式 gitbook mobi ./ ./mybook.mobi
